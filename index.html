Télécharger PRIMA App V0.1 — ZIP portable autonome

Version test portable PRIMA — aucune installation nécessaire. Télécharge le ZIP, extrais-le, puis double-clique sur LANCER_PRIMA.bat.

Prérequis : Windows 10 ou 11 uniquement. Pas de Node.js, pas d’AutoHotkey, pas d’installateur à lancer.

Installation (3 étapes)

  1. Télécharge le fichier PRIMA_App_V0_1_Family_Portable_NoNodeInstall.zip.
  2. Extrais le ZIP dans un dossier (ex. C:\PRIMA_Test) — ne lance pas le ZIP directement.
  3. Ouvre le dossier extrait et double-clique sur LANCER_PRIMA.bat.

Note Windows : si SmartScreen s’affiche, clique « Informations complémentaires » puis « Exécuter quand même ». Version locale non signée.

Cette version (V1 famille)

  • PRIMA Génie — parcours unique (Soul → VDA → Shadow)
  • Choix multiples — plusieurs réponses possibles (compteur visible)
  • Réponds instinctivement, sans te censurer
  • Lecture symbolique — pas un diagnostic médical
  • Reprise sécurisée · boutons verrouillés si non disponibles
  • Profils et données restent sur votre ordinateur

Important : créez un profil neuf pour ce test.

Parcours test

  1. Lance LANCER_PRIMA.bat → PRIMA s’ouvre.
  2. Crée un profil neuf → va sur PRIMA Soul → PRIMA Génie.
  3. Réponds aux questions (une ou plusieurs réponses) → génère ta lecture.
  4. Appelle Kelly après avoir vu tes résultats.

Si Génie ne démarre pas : envoie logs\genie_engine_last_error.txt du dossier extrait.
