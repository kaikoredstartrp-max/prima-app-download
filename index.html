Téléchargement

Télécharger PRIMA App

Fichier : PRIMA_App_V0_1_Install.zip (sans profils ni données personnelles).

Installation

  1. Téléchargez le fichier ZIP.
  2. Extrayez le dossier.
  3. Installez AutoHotkey v2 si demandé (autohotkey.com).
  4. Lancez launch_prima.ahk ou PRIMA App.lnk.
  5. Créez votre profil et choisissez un mot de passe.
  6. Utilisez Kit PRIMA pour vérifier l'installation.

À savoir

  • AutoHotkey v2 requis (pas la version 1).
  • Fonctionnement local : vos données restent sur votre PC.
  • Aucun compte cloud ni clé API obligatoire.
