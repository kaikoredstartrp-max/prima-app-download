PRIMA App V0.1 — Installation
=============================

PRIMA App est une version locale. Vos profils restent sur votre ordinateur.

Contenu :
- PRIMA App.exe (application Windows avec logo PRIMA)
- Soul / VDA / Shadow
- Questions chirurgicales
- Ranking optionnel : minimum 1 réponse par question (non choisies = 0 point)
- Résultats en pourcentages réels, boussole PRIMA, export lecture
- Reprise test sécurisée, rester connecté sécurisé
- Choix Univers, Territoire, Race / Forme symbolique
- Identité PRIMA (3 choix visibles) verrouillée par profil

Important :
Pour cette version, créez un profil neuf.
Les anciens profils de test peuvent contenir des états corrompus.

Installation (installateur Windows) :
1. Téléchargez PRIMA_App_Setup.exe.
2. Lancez l'installateur.
3. Suivez l'assistant (aucun droit administrateur requis).
4. Ouvrez PRIMA App depuis le bureau ou le menu Démarrer.
5. Créez un profil neuf et un mot de passe local.

Message Windows SmartScreen
===========================

Au premier lancement, Windows peut afficher :
« Windows a protégé votre ordinateur ».

Cela arrive parce que PRIMA App V0.1 n'est pas encore signée avec un certificat officiel.

Pour lancer l'app :
1. Cliquer sur « Informations complémentaires »
2. Cliquer sur « Exécuter quand même »

Important :
Téléchargez PRIMA App uniquement depuis le lien officiel GitHub Pages.

AutoHotkey v2 n'est pas requis : l'application est fournie en .exe.

Version portable (ZIP) :
Une archive ZIP reste disponible en téléchargement secondaire pour usage sans installateur.
