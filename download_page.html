Installation

  1. Téléchargez le fichier ZIP.
  2. Extrayez le dossier PRIMA App.
  3. Double-cliquez sur PRIMA App.exe.
  4. Créez un profil neuf et un mot de passe.

Ne lancez pas l’application depuis le ZIP non extrait. AutoHotkey v2 n’est pas requis.
