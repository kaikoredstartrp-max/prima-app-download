Télécharger l’installateur PRIMA — aucune installation technique requise.

Fichier : PRIMA_App_Setup.exe (~25 Mo). Installateur Windows avec runtimes embarqués (AutoHotkey v2, Node portable, moteur CJS). Aucun Node.js ni AutoHotkey à installer séparément.

Lance l’installateur, puis ouvre PRIMA App depuis le menu Démarrer ou le raccourci bureau.

Prérequis : Windows 10 ou 11 uniquement.

Note Windows : si SmartScreen s’affiche, clique « Informations complémentaires » puis « Exécuter quand même ».

Installation

  1. Télécharge PRIMA_App_Setup.exe.
  2. Lance l’installateur et suis les étapes.
  3. Ouvre PRIMA App (menu Démarrer ou raccourci bureau).

Cette version (V1 famille)

  • PRIMA Génie — parcours unique (Soul → VDA → Shadow)
  • Choix multiples — plusieurs réponses possibles (compteur visible)
  • Réponds instinctivement, sans te censurer
  • Lecture symbolique — pas un diagnostic médical
  • Profils et données restent sur ton ordinateur
