Installation

  1. Téléchargez le fichier ZIP.
  2. Extrayez le dossier.
  3. Installez AutoHotkey v2 (autohotkey.com).
  4. Lancez launch_prima.ahk.
  5. Créez un profil neuf et un mot de passe.
  6. Utilisez Kit PRIMA pour vérifier l'installation.
